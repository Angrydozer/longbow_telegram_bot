# longbow_telegram_bot

Данный телеграм бот позволяет записывать повторяющиеся действия в data.csv (с целью длительного сбора информации и последующего анализа) с помощью всплывающих клавиатур

Для этого предварительно собирается конфигурация (config.csv) с типами действий и вариантами ответов, откуда и подтягивается информация по клавишам в клавиатурах
