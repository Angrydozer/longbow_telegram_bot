# longbow_telegram_bot

Данный бот bot.py позволяет записывать поторяющиеся действия в таблицу, быстро и удобно через выскакиваемые клавиатуры

Для этого предварительно составляется конфигурация запрашиваемой информации в виде config.csv, откуда и подтягивается инфа по клавишам в клавиутурах

Данные записываются в data.csv
