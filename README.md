# longbow_telegram_bot

Данный телеграм бот позволяет записывать повторяющиеся действия в data.csv (удобно при длительном сборе часто повторяющихся действий и дальнейшем анализе данных)

Скорость записи осуществляется помощью всплывающих клавиатур с прописанными заранее или популярными ответами

Для этого предварительно собирается конфигурация (config.csv) с типами действий и вариантами ответов, которая определяет какие данные пользователь сможет записывать
